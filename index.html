Pertanyaan Umum

Ada yang Bingung?

Jawaban untuk pertanyaan yang paling sering ditanya mahasiswa.

Sangat mudah! Cukup 4 langkah: (1) Upload file dokumenmu (PDF, DOC, DOCX). (2) Pilih opsi cetak — ukuran kertas, warna, jumlah salinan, dan lokasi printer. (3) Bayar via QRIS atau transfer. (4) Tunggu notifikasi WhatsApp bahwa dokumenmu sudah siap diambil!
SmartPrint saat ini mendukung format: PDF, DOC, dan DOCX. Ukuran file maksimal 5 MB per file agar proses upload dan cetak tetap cepat dan stabil.
Harga dihitung otomatis berdasarkan pilihan kamu:

• Hitam Putih: Rp 1.000/halaman
• Berwarna: Rp 2.000/halaman
• Total: halaman × salinan × harga/halaman

Biaya finishing ditampilkan terpisah pada rincian biaya sebelum pembayaran.
Ya, keamanan file adalah prioritas kami. File kamu disimpan di Supabase Cloud Storage yang terenkripsi dengan standar industri. File hanya bisa diakses oleh admin untuk keperluan cetak, dan otomatis dihapus setelah pesanan selesai. Tidak ada pihak ketiga yang dapat mengakses filenya.
Klik tombol 📦 Status di bagian atas website, lalu masukkan ID pesananmu (format: SP-XXXX-YYYY). Kamu bisa melihat timeline lengkap: Pending → Konfirmasi → Mencetak → Selesai. Kamu juga akan otomatis mendapat notifikasi WhatsApp setiap kali status berubah.
Rata-rata kurang dari 5 menit untuk dokumen biasa (1–20 halaman). Untuk dokumen besar atau dalam kondisi antrian penuh, estimasi waktu ditampilkan saat kamu melihat status pesanan. Sistem nomor antrian digital memastikan pesananmu diproses sesuai urutan.
Tidak perlu! Kamu bisa langsung memesan tanpa akun. Namun dengan akun, kamu mendapat keuntungan: data nama & WhatsApp auto-terisi, riwayat semua cetakan tersimpan, dan lebih mudah cek pesanan. Daftar gratis lewat tombol 👤 Akun di navbar.

Konfigurasi
Setup Supabase

Gratis selamanya · File & pesanan tersimpan di cloud

1️⃣ Buat akun Supabase

Buka supabase.com → Daftar gratis → Buat project baru → pilih region Southeast Asia

2️⃣ Buat tabel orders

Di Supabase → SQL Editor → jalankan SQL ini:

create table orders ( id uuid default gen_random_uuid() primary key, order_id text unique not null, created_at timestamptz default now(), nama text, whatsapp text, doc_type text, file_name text, file_url text, bukti_url text, paper text, color text, side text, pages int, copies int, finishing text, doc_info jsonb, total int, status text default 'pending' ); alter table orders enable row level security; create policy "anon insert" on orders for insert to anon with check (true); create policy "anon select own" on orders for select to anon using (true); create policy "anon update own" on orders for update to anon using (true);

3️⃣ Buat Storage Bucket

Storage → New Bucket → nama: smartprint-files → centang Public bucket → Save

4️⃣ Buat Tabel Users

SQL Editor → paste → Run:

create table if not exists users (
  id bigint generated always as identity primary key,
  nama text, email text unique not null,
  wa text, pw_hash text,
  created_at timestamptz default now()
);
alter table users enable row level security;
create policy "users_all" on users for all to anon using (true) with check (true);

4️⃣ Masukkan kredensial

Settings → API → copy Project URL dan anon public key

Kredensial disimpan di browser kamu saja · Aman
